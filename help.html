❓ How It Works

Lost something?

Post it on the Lost page with a secret identifying detail. If someone finds it and claims it, you'll see their name on your card.

Found something?

Post it on the Found page with a verification question. Only the true owner will know the answer.

Verification system

Claims require answering a secret question set by the poster. This prevents false claims and ensures items go to their rightful owners.
